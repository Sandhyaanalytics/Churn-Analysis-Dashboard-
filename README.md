# Churn-Analysis-Dashboard-
Analysing the Customer Churn

Customer Churn Analysis Dashboard

Project Overview

This project analyzes customer churn behavior using data visualization in Power BI.
The goal is to identify key factors driving customer churn and provide actionable business recommendations to improve retention.

---

Objectives

- Analyze customer churn patterns
- Identify high-risk customer segments
- Understand impact of contract type, tenure, and support tickets
- Provide business recommendations to reduce churn

---

Tools Used

- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling

---

Dashboard Features

Key KPIs

- Total Customers
- Churned Customers
- Churn Rate (%)
- Average Tenure (Months)
- Average Monthly Fee

---

Key Visuals

- Churn by Contract Type
- Churn by Subscription Type
- Churn vs Tenure
- Churn vs Support Tickets
- Churn by Payment Method

---

Key Business Insights

- Customers on monthly contracts show higher churn compared to long-term contracts
- Customers with shorter tenure are more likely to churn
- Higher number of support tickets is associated with increased churn
- Basic subscription users show higher churn compared to premium users

---

Business Recommendations

- Encourage customers to switch to long-term contracts through incentives
- Improve onboarding experience to retain new customers
- Enhance customer support services to reduce dissatisfaction
- Add more value to basic subscription plans to improve retention
